Core Features

핵심 기능

01

대시보드

주요 지표를 한눈에 파악할 수 있는 대시보드 화면입니다. 차트, 수치, 상태 카드를 통해 서비스 현황을 빠르게 모니터링합니다.

  • KPI 요약 카드
  • 실시간 차트 레이아웃
  • 최근 활동 피드
02

사용자 관리

회원 목록 조회, 권한 설정, 상태 변경 등 사용자 관련 작업을 수행하는 페이지입니다. 필터와 검색 기능으로 빠른 조회를 지원합니다.

  • 테이블형 목록 + 필터
  • 권한 그룹 관리
  • 상태별 배지 표시
03

데이터 관리

서비스에서 발생하는 데이터를 조회하고 관리하는 화면입니다. 엑셀 내보내기, 상세 필터, 페이지네이션을 지원합니다.

  • 고급 필터 패널
  • 인라인 편집
  • 엑셀 내보내기
04

설정 / 권한 관리

시스템 설정 및 역할별 접근 권한을 설정하는 페이지입니다. 세분화된 메뉴 접근 권한과 기능 단위의 통제가 가능합니다.

  • 역할(Role) 기반 권한 설정
  • 메뉴 / 기능별 접근 제어
  • 변경 이력 로그
